# Todo List App

This project was created for **Week 1 of the Code the Dream (CTD)** program.  
It is a simple React + Vite application that displays a title and an unordered list of todos.

## Installation

```bash
git clone https://github.com/deryakendircikahraman/todo-list-ctd.git
cd todo-list
npm install
Run the Development Server
npm run dev
Then open the local URL shown in the terminal (http://localhost:5173).

Project Overview
This project fulfills the Week 1 requirements:
Created using Vite’s React template with JavaScript
Cleaned up default starter files
```
